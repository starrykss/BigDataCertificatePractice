# Repository for Certificate Big Data Analysis Engineer(빅데이터분석기사)

## 레포지토리 소개
- **빅데이터분석기사** 실기 시험을 준비하면서 공부한 내용들을 정리한 레포지토리

## 레포지토리 구성
- `./study/` : 공부 내용 정리 (Python, NumPy, Pandas 등)
- `./datasets/` : 실습 데이터 세트
- `./src/` : 예제 파일
- `./res/` : 정답 파일

## 참고 자료
